# Tutorial 3
Di tutorial ini, saya memberikan animasi walk, jump, double jump, dan fall. Dengan menggunakan `sprite.play` dan condition yang tepat, animation akan berjalan sesuai dengan kondisi player.  Untuk mengatasi masalah animasi 'overflow' saya menggunakan `sprite.stop()`, dan untuk arah animasi saya menggunakan `sprite.flip_h`.

[Reference untuk `sprite.stop()`](https://forum.godotengine.org/t/how-do-you-stop-animatedsprite2d-once-animation-is-down/57438)
